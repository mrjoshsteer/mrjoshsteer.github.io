# mrjoshsteer.github.io

Mr Josh Steer's Personal Website

A tribute website built for Mr Josh Steer, a maths, economics, and business
studies teacher, created and maintained by one of his former students as a
personal keepsake.

## Live Site

https://mrjoshsteer.github.io

## About

This is a fan-made tribute site; it is not officially run by Mr Steer
himself. It includes a short bio, an overview of his teaching approach, a
small photo gallery, a birthday countdown, and a link to his LinkedIn
profile.
